Welcome to the Dats doc!

This repo contains the tex sources which are used to generate the Dats
document. It contains the tutorial (see section "The basics"), specifies
the semantics and syntax, and describes the workings of the Dats language.

To obtain a copy of the PDF, go to:
```
https://texlive2020.latexonline.cc/compile?git=https://github.com/dats-lang/dats-tex.git&target=dats-2.tex&command=pdflatex&force=true&download=dats-2.pdf
``
